# きょうのよていボード

知的障害のある子どもが、今日1日の予定を絵カードで理解しやすくするための、HTMLだけで動く予定表アプリです。

## 構成

- `index.html`：アプリ本体
- `cards/`：絵カード画像フォルダ

## 使い方

1. GitHubリポジトリに `index.html` と `cards` フォルダをそのまま置く
2. GitHub Pagesを有効化する
3. ブラウザで開く
4. 枠をクリックして絵カードを選ぶ
5. 必要に応じて「保存」を押す

## 画面仕様

- 3段構成
  - あさ
  - ひる
  - よる
- 各段8枠
- 枠をクリックすると絵カード選択画面が開く
- 選んだカードがその枠に貼られる
- カードの差し替え・削除が可能
- 「できたチェック」をONにすると、予定完了チェックが可能

## 保存仕様

ブラウザの localStorage に保存します。
同じ端末・同じブラウザでは次回も読み込めます。
